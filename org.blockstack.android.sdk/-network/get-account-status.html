getAccountStatus

fun getAccountStatus(address: String, tokenType: String, callback: (Result<AccountStatus>) -> Unit): Unit

Get the status of an account for a particular token holding. This includes its total number of expenditures and credits, lockup times, last txid, and so on.

Parameters

address - the account's address

tokenType - the token type to query

callback - called with a result object that contains the state of the account for this token or error if the request failed
