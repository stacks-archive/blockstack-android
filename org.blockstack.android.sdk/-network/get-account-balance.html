getAccountBalance

fun getAccountBalance(address: String, tokenType: String, callback: (Result<BigInteger>) -> Unit): Unit

Get the number of tokens owned by an account. If the account does not exist or has no tokens of this type, then 0 will be returned.

Parameters

address - the account's address.

tokenType - the type of token to query.

callback - called with a result object that contains the number of tokens held by this account in the smallest denomination.
