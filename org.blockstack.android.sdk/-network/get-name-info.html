getNameInfo

fun getNameInfo(fullyQualifiedName: String, callback: (Result<NameInfo>) -> Unit): Unit

Get WHOIS-like information for a name, including the address that owns it, the block at which it expires, and the zone file anchored to it (if available).

Parameters

fullyQualifiedName - the name to query. Can be on-chain of off-chain.

callback - called with a result object that contains the WHOIS-like name information or error if the request failed.
